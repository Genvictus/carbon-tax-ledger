# Carbon Tax Ledger
A transparent and immutable ledger for carbon tax transactions, built with Hyperledger Fabric.

## System requirements
- Linux or WSL
- Docker

## Installation
1. Run `./setup.sh` to start the network, smart contract, frontend, backend, and oracle.
